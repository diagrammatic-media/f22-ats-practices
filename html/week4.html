Create an interface with a "button" and a dynamic background.

button

this is not the "button" feature in p5*js -- don't use that reference material. Instead you will draw a shape (Use square, circle, triangle, or rectangle. Irregular shapes are not recommended!) that will serve as your button. Use an if/else statement to change the color of the button if your mouse is on top of the shape. (BONUS: create a second color change if the mouse is pressed using the system variable mouseIsPressed?.

background

Use a second if/else statement to change the color of the background when the mouse is in a certain region of the canvas.

Tips
